A responsive web app that converts images into customizable ASCII art with support for color, presets, PNG transparency, and embeddable output code.

## ‚ú® Features

- üé® **Custom ASCII Characters**: Use your own set of characters or select a preset style (`blocky`, `smooth`, `detailed`, etc.)
- üìè **Adjustable Output Size**: Set custom `width` and `height`, or use `"auto"` to preserve image ratio.
- üåà **Color Mode**: Toggle colored output based on pixel color or switch to monochrome.
- üßº **Transparent PNG Support**: Automatically skips pixels with `alpha = 0` in PNGs.
- üì± **Device Optimized Output**: Select your device (Desktop, Tablet, Mobile) for best-fit font size.
- üß© **Embeddable Code Generator**: Create styled HTML snippets with options like:
  - `Default` size
  - `Fullscreen` (fits entire device screen)
  - `Half-screen` (fits half of the screen)
- üíæ **Export**: Download ASCII result as HTML.

---

## üõ†Ô∏è Usage

1. **Upload Image**: Choose `.png`, `.jpg`, or `.jpeg` files.
2. **Adjust Options**:
   - Width & Height (`auto` or number)
   - ASCII Characters (e.g., `@#S%?*+;:,.`)
   - Color output: On/Off
   - Style Presets
   - Device type: Desktop, Tablet, or Mobile
3. **View ASCII Art** in the result box.
4. **Copy or Download** the generated HTML/CSS from the `Embeddable Code` section.
